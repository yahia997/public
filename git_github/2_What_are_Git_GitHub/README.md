# What are Git & GitHub ?
Everyone is talking about Git and GitHub. Here we will explain:
1. What are Git & GitHub?
2. Differences between Git & GitHub

As we explained in the previous phase ("What is a VCS (Version Control System) ?"). Git is actually a VCS (not the only one in the market, but it is the most widely used, all of which have the same idea).

Because Git is a VCS, it has the same functions:
- Saves your history of modifications.
- Efficient Collaboration from other developers.

Now, what is GitHub?
Imagine you and your friend are working on the same project simultaneously. You are adding your modifications, and he/she does also. So both of you have a different version. Here, GitHub helps to merge the differences into a single project.
Think of GitHub as an online platform to store your project and push modifications to it; you can give other developers access to contribute to the project.
